🎬 Video Skip Buttons DLC

Sistema de botones de salto de tiempo para videos embedidos

📖 ¿Cómo funciona?

El sistema detecta automáticamente videos con parámetros especiales y agrega botones de salto:

?d=90&op=00:00&ed=03:20
  • d: duración para rangos implícitos (opcional, default: 90s)
  • op: tiempo para Opening (MM:SS, [MM:SS], [MM:SS-MM:SS] o MM:SS+90s)
  • ed: tiempo para Ending (MM:SS, [MM:SS], [MM:SS-MM:SS] o MM:SS+90s)

🎯 Lógica de Corchetes

Regla simple:

  • Si el dato está entre corchetes [...] → Usa duration (rango implícito)
  • Si el dato no está entre corchetes → Es punto fijo

🎛️ Controles Manuales

🎵 Video con OP y ED (Puntos Fijos)

Parámetros: op=00:30&ed=03:20 (puntos fijos)

🎬 Video con OP como Rango Implícito

Parámetros: d=120&op=[01:30] (rango: [01:30-03:30])

🎭 Video con ED como Rango Exacto

Parámetros: ed=[02:00-03:30] (rango exacto)

⏱️ Video con Duración Explícita

Parámetros: op=01:00+45s (rango: [01:00-01:45])

🎬 Video solo con duración

Parámetros: d=120 (solo duración, sin botones)

🎹 Video sin parámetros (normal)

Sin parámetros (funciona como video normal, sin botones)

📹 Video HTML5 nativo

Parámetros: d=60&op=[00:10]&ed=00:40

🔧 Combinación Completa

Parámetros: d=90&op=[01:30]&ed=[22:00-23:30]

🚀 Características del DLC

  • Detección automática de videos con parámetros especiales
  • Botones flotantes con diseño moderno y responsivo
  • Soporte para YouTube, videos HTML5 y otros servicios
  • Lógica de corchetes: [rango] vs punto fijo
  • Duration flexible con parámetro d=
  • Integración con ∞Iframes Matrix Viewer
  • Animaciones suaves y notificaciones visuales
  • Detección automática de nuevos videos via MutationObserver
  • API pública para control programático
  • Compatibilidad con navegadores modernos
  • Código modular y extensible

🔧 Uso Programático

// Agregar un video manualmente window.videoSkipButtons.addVideo(elementoVideo, 'd=120&op=[01:30]&ed=02:00'); // Actualizar configuración de un video window.videoSkipButtons.updateVideoConfig('video-id', { op: 60, ed: 180 }); // Destruir el DLC window.videoSkipButtons.destroy();
